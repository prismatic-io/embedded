## Internationalization (i18n)

Translate any string in embedded marketplace to match your product's voice or locale. The `debug` flag below is on, which renders each translatable string as its translation key — so you can see exactly what to override:

```
"activateIntegrationDialog.banner.text--isNotConfigurable":
  { _: "Veuillez contacter %{organization} pour activer cette intégration" }
```

Your custom content — integration names, config variable names, wizard titles — is matched by the exact string instead of a translation key, so include any HTML the source contains (e.g. `"<h1>My Text</h1>": "<h1>New Text</h1>"`).

Full reference: [Translations and Internationalization](https://prismatic.io/docs/embed/translations-and-internationalization/).
