## Workflow Builder

Enable customers to build their own automations — approvals, routing rules, data transforms — without forcing that logic into your shared integrations. Workflows run on the same runtime as your integrations, so you reuse connectors, logs, and execution history instead of shipping a second product.

Rendered with `showWorkflows()`. Set `screenConfiguration.workflows.includeIntegrations` to `true` to list integrations alongside workflows.

Full configuration reference: [Embedded Workflow Builder](https://prismatic.io/docs/embed/workflow-builder/workflow-builder/).
